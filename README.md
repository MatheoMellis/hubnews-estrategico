# HUBNEWS | Inteligência Estratégica 2026

Bem-vindo ao **HUBNEWS**, um dashboard de notícias com um design clean e profissional (*Bento Box UI*), focado em fornecer informações filtradas e de alto valor.

Este projeto possui rotas e filtros dinâmicos segmentados por categorias, pensado perfeitamente para profissionais de TI, concurseiros e entusiastas de IA.

## 🗂️ Categorias Abrangentes
*   **Concursos TI:** Sub-dividido em *Visão Geral*, *Bancos* (BB, Caixa), *Setor Federal* (BNDES, TSE), *Carreira Policial* (PF, PC) e *Faculdades*.
*   **Inteligência Artificial:** Sub-dividido em *IAs Locais (Edge)*, *IAs na Nuvem (APIs)* e *Ferramentas Agênticas*.
*   **Atualidades:** Sub-dividido em *Brasil*, *Mundo* e *Economia*.

## 🚀 Como Executar Localmente
1.  O servidor de desenvolvimento já está configurado.
2.  Acesse `http://localhost:5175/` (ou a porta informada pelo Vite).

## ☁️ Preparado para GitHub e Vercel
Este projeto foi reconstruído usando **Vite**, **React** e **TailwindCSS**, sendo uma SPA (Single Page Application) puramente estática. Isso permite hospedagem gratuita e extremamente rápida na Vercel.

### Passo 1: Subir para o GitHub
\`\`\`bash
git init
git add .
git commit -m "feat: initial commit with bento dashboard"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/app-de-noticias.git
git push -u origin main
\`\`\`

### Passo 2: Hospedar na Vercel (Gratuito)
1. Crie uma conta em [vercel.com](https://vercel.com/) (você pode fazer login com o seu GitHub).
2. Clique em **Add New Project**.
3. Importe o repositório `app-de-noticias` que você acabou de criar no GitHub.
4. A Vercel detectará automaticamente que é um projeto **Vite**.
5. Clique em **Deploy**.
6. Em menos de um minuto, seu painel de notícias estará online para o mundo!

## 🛠️ Tecnologias
- **React + Vite**: Performance e rapidez de desenvolvimento.
- **Tailwind CSS v4**: Design minimalista e profissional, com tipografia *Inter*.
- **Framer Motion**: Animações de interface.
- **Lucide React**: Ícones minimalistas.
